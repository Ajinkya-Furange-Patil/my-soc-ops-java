# 🎲 Soc Ops

> **The ultimate icebreaker game for in-person events.** Find people who match the questions, mark your card, and get 5 in a row to win!

Turn boring mixers into memorable connections with **Soc Ops**—a modern twist on classic bingo designed to get people talking, laughing, and networking.

---

## ✨ What Makes Soc Ops Special

- **Instant Connection** – Real questions that spark genuine conversations
- **Fast & Fun** – Rounds complete in 10-15 minutes
- **Works Anywhere** – Web-based, runs on any device
- **Easy to Host** – Start a game with one click
- **Customizable** – Add your own questions and themes
- **Built with AI** – Developed collaboratively using GitHub Copilot

---

## 🛠️ Tech Stack

**Backend:** FastAPI | **Frontend:** Jinja2 + HTMX | **Styling:** Custom CSS Utilities

No build step. No JavaScript frameworks. Just fast, lean, and responsive.

---

## 🚀 Quick Start

### Try It Now (No Setup Required)
Visit the [live demo](https://copilot-dev-days.github.io/agent-lab-python/) to see Soc Ops in action.

### Run Locally (5 minutes)
```bash
# Clone & install
git clone https://github.com/copilot-dev-days/agent-lab-python
cd agent-lab-python
python -m pip install -e .

# Start the game server
uvicorn app.main:app --reload --port 8000

# Open http://localhost:8000
```
